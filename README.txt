* **Homepage Settings** - Customize page title, meta description, and H1 tag for your homepage

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to **Plugins > Add New**
3. Search for "Complete SEO Control"
4. Click **Install Now** and then **Activate**

= Manual Installation =

1. Download the plugin ZIP file
2. Log in to your WordPress admin panel
3. Navigate to **Plugins > Add New > Upload Plugin**
4. Choose the ZIP file and click **Install Now**
5. Click **Activate Plugin**

= After Activation =

1. Navigate to **SEO Control** in your WordPress admin menu
2. Start with the **Homepage** tab to set your site's main SEO settings
3. Use the other tabs to optimize individual posts, pages, and categories

== Frequently Asked Questions ==

= Will this plugin slow down my site? =

No. Complete SEO Control is built with performance in mind. It uses efficient database queries and only loads its admin assets on the plugin's admin pages.

= Is this compatible with my theme? =

Yes! Complete SEO Control works with any properly coded WordPress theme. It uses standard WordPress functions and hooks.

= Can I use this with other SEO plugins? =

While technically possible, we recommend using only one SEO plugin to avoid conflicts and duplicate meta tags.

= Will I lose my data if I deactivate the plugin? =

No. Your data remains in the database if you deactivate the plugin. However, if you delete/uninstall the plugin, all data will be removed.

= How do I add custom SEO to a new post or page? =

Simply create your post/page, then go to the SEO Control plugin, navigate to the Articles or Pages tab, search for your content, and click "Edit SEO" to customize.

= What happens if I don't set custom SEO? =

The plugin will use default values based on your content's title. Your site will still function normally.

= Does this plugin add social media meta tags? =

Not in the free version. Social media meta tags (OpenGraph, Twitter Cards) are planned for the premium version.

= Is the plugin translation-ready? =

Yes! Complete SEO Control is fully internationalization-ready and can be translated into any language.
